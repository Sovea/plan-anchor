# Intake Checklist

- [ ] State the mission in one sentence.
- [ ] Separate scope from non-goals.
- [ ] Identify constraints that could change the solution.
- [ ] Convert vague expectations into observable acceptance criteria.
- [ ] Identify risks and unknowns.
- [ ] Ask focused questions for any missing item that could affect implementation.
- [ ] Avoid planning implementation details until critical ambiguity is resolved.
