# Pre-Completion Checklist

- [ ] Every acceptance criterion is satisfied or explicitly out of scope.
- [ ] Every Work Unit is complete or intentionally deferred with a reason.
- [ ] Static checks have passed or gaps are disclosed.
- [ ] Tests have passed or gaps are disclosed.
- [ ] Build has passed when relevant or gaps are disclosed.
- [ ] Behavioral verification has been performed when relevant or gaps are disclosed.
- [ ] Drift log is empty, resolved, or approved.
- [ ] Durable anchor is updated when durable state is enabled.
- [ ] No durable anchor file is created or committed unless explicitly approved.
- [ ] No hidden TODO-only behavior, partial implementation, or dead path remains.
- [ ] Final response distinguishes completed work from remaining work.
