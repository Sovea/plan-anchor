# Context Compaction Checklist

Use this before context compaction, long pauses, or task switching when governed work may continue later.

- Restate the Mission Contract in 1-3 bullets.
- List acceptance criteria with current status and evidence.
- List completed Work Units with evidence.
- Identify the active Work Unit and exact smallest next action.
- Record remaining Work Units or acceptance criteria.
- Record verification status, including checks not run and why.
- Record drift decisions and unresolved drift.
- Record blockers and unresolved user questions.
- Record repository assumptions, including branch and working tree status if known.
- If durable state is enabled, update the durable anchor before handoff.
- Produce Handoff State that a fresh Claude instance can use without reading the full conversation.
